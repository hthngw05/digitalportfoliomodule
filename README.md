# Ho Tran Hoai Thuong — Digital Portfolio

Responsive nine-page portfolio prepared for the CIT2C27 Guided Work-Based Learning 1 assessment. It includes three workplace project case studies, two downloadable digital artefacts, TSC/CCS competency mapping, actions and results, training records, and a future-work-self reflection.

## Before publishing

1. Confirm the personal and project descriptions are accurate.
2. Obtain permission from people visible in any group photograph before public publication, or remove the group image.
3. Do not add internal screenshots, operational data or production identifiers without written company clearance.
4. Correct the internship end date in the separate OJT Learning Outcomes form from 31 March 2026 to 31 March 2027 through the appropriate school/company contact.

## Run locally

Open `index.html` in a browser. For a local web server, run:

```bash
python -m http.server 8000 --directory portfolio
```

Then visit `http://localhost:8000`.

## Publish with GitHub Pages

Create a repository, upload the contents of this folder, and enable Pages under **Settings → Pages** using the main branch and root folder. Do not publish until all confidential information has been removed and the evidence has been approved.
